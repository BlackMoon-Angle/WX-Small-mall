# 电商后台管理系统

## 技术栈：Vue + VueRouter + Axios + Element UI + NodeJs + MySQL

## 主要功能：

### 				用户管理

​						用户列表

### 				权限管理

​						角色列表

​						权限列表

### 				商品管理

​						商品列表

​						分类参数

​						商品分类

### 				订单管理

​						订单列表

### 				数据统计

​						数据列表
