# 微信小小商城

## 技术栈：微信框架

## 主要功能：

### 				首页

​						首页的轮播图、导航跳转

### 				分类

​						商品分类

​						商品跳转

​						商品详细

### 				购物车

​						商品的增删改

### 				个人中心

​						登录

​						查看收藏商品

​						查看订单（需要授权）
​						查看付款（需要授权）

​						意见反馈
